sftools

Bunch of tools for SFML 2.0 application development.

Like SFML, sftools works on Windows, Linux and Mac OS X.

Downloads

Pre-Alpha version : sftools 2.0 is still in development, some feature will be added and some might be modified. Keep that in mind.

Download .zip Download .tar.gz

Installation

sftools is a header-only library. Thus the only thing your need to do to use it in your code is to download the source and extract them to, let's say some/path/ and tell your compiler to look for header at some/path/include/. With most compiler you can use the -I.

Or you can drop the content of include/ folder to /usr/local/include, or any other standard path of your compiler/OS.
